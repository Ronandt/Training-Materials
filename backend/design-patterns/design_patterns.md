## Adapter Method

Adapter acts as a translation layer of 2 seperate functions. This can help join your code with external dependencies, or act as a connector between 2 functions that were designed seperately. When learning coding, you intuitively will learn this to simplify existing code and reducing the need to tailor each function to meet the requirements of the imported functions.

**Example**: In this case, I want to pass classes into a train() function, but the classes have seperate requirements.
